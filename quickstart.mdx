## Start Using Zeepay

To start using Zeepay's APIs, please reach out to [info@myzeepay.com](mailto:info@myzeepay.com) for onboarding. Once onboarded, you can integrate our APIs seamlessly into your applications..

### Onboarding Steps

<Tip>
    1. Reach out to [info@myzeepay.com](mailto:info@myzeepay.com) for onboarding.
    2. Obtain your API keys.
    3. Explore the documentation.
    3. Start testing in the sandbox.

</Tip>

### Send Money

<AccordionGroup>
  <Accordion icon="rectangle-terminal" title="Sample Request Body">
  <Info>
    **Note**: Authenticate all Zeepay API calls using your secret keys.
    ```json
      curl {{base_url}}/payouts 
      -H "Authorization: Bearer YOUR_SECRET_KEY"
      -H "Content-Type: application/x-www-form-urlencoded"
      -X POST
    ```
  </Info>
