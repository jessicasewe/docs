---
title: "Zeepay Response Codes"
description: "The Zeepay APIs return a standard set of status codes that cover various success, failure, and pending scenarios. These codes and their corresponding scenarios are outlined below."
---

#### Response Codes

<Steps>
  <Step title="MFL - Master Wallet Funds Low" icon="map-pin">
    No action Required
  </Step>
  <Step title="TSE - Transaction Service Type Empty" icon="map-pin">
    Zeepay Support to communicate with MTO if action is required.
  </Step>
  <Step title="TME - Transaction MNO Empty" icon="map-pin">
    Transaction MNO field is missing. Zeepay Support to engage the MTO if necessary.
  </Step>
  <Step title="PSFL - Partner Sub-Wallet Funds Low" icon="map-pin">
    Zeepay Support to communicate with MTO if action is required.
  </Step>
  <Step title="DTGP - Dispatched to Payment Gateway" icon="map-pin">
    No action required from MTO.
  </Step>
  <Step title="DWRL - Destination Wallet Resolution Failure" icon="map-pin">
    Invalid or missing receiving country
  </Step>
  <Step title="OADF - OVA (Zeepay) Debit Failure" icon="map-pin">
    No action required from MTO.
  </Step>
  <Step title="RCF - OVA (Zeepay) Debit Failure" icon="map-pin">
    No action required from MTO.
  </Step>
  <Step title="PACRF - Partner Agency Code Resolution Failure" icon="map-pin">
    Aggregator Related Only.
  </Step>
  <Step title="SWRF - Sub-Wallet Resolution Failure" icon="map-pin"></Step>
  <Step title="ACF - Authentication Code Failure" icon="map-pin">
    Aggregator Related Only.
  </Step>
  <Step title="SCF - Sending Country resolution failure" icon="map-pin">
   Invalid or missing sending country
  </Step>
  <Step
    title="TAF - Transaction Amount Resolution Failure"
    icon="map-pin"
  >Insufficient or Missing Amount</Step>
  <Step title="WIVF - Transaction Amount Resolution Failure" icon="map-pin">
    Aggregator Related Only.
  </Step>
</Steps>
