## API Basics

The Zeepay API provides access to a wide range of features, enabling you to integrate and extend them seamlessly within your application. It is designed to be RESTful and is structured around key resources for efficient interaction.

## Authentication

All API endpoints are authenticated using Bearer tokens and picked up from the specification file.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
